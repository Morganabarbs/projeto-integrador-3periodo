# Backend

API responsável pela lógica de negócio da aplicação, desenvolvida em Java com Spring Boot.

## 🚀 Responsabilidades
- Processamento das regras de negócio
- Exposição de endpoints REST
- Integração com banco de dados MySQL
- Integração com serviços externos (OCR e e-mail)

## 🛠️ Tecnologias
- Java
- Spring Boot
- MySQL

## 📡 Integração
Este backend é consumido pelos frontends:
- Web (React PWA)
- Mobile (React Native)
