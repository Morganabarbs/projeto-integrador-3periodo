# Frontend Web

Aplicação web desenvolvida em React, com suporte a Progressive Web App (PWA), responsável pela interface do usuário no navegador.

## 🚀 Funcionalidades
- Interface interativa para usuários
- Consumo da API do backend
- Suporte a funcionamento como aplicativo (PWA)

## 🛠️ Tecnologias
- React
- PWA

## 📡 Integração
Este frontend se comunica com o backend via API REST.
